### 圆点位置计算（优化后）
```javascript
// 圆心: (60, 60)
// 半径: 45
// 角度: π * (1 - value/100)

cx = 60 + 45 * Math.cos(Math.PI * (1 - value / 100))
cy = 60 - 45 * Math.sin(Math.PI * (1 - value / 100))
```

## 📊 色段间隙设计

### 间隙位置
```
     ╭─间隙─╮
    ╱         ╲
   │  红 橙 黄 绿  │
   │  ↑ ↑ ↑ ↑   │
   │  │ │ │ │   │
    ╲ 间隙间隙间隙╱
     ╰─────────╯
```

### 角度分布
- **Segment 1 (红色)**: 180° → 135° (0-25)
- **间隙 1**: 135° → 132°
- **Segment 2 (橙色)**: 132° → 90° (25-50)
- **间隙 2**: 90° → 87°
- **Segment 3 (黄色)**: 87° → 45° (50-75)
- **间隙 3**: 45° → 42°
- **Segment 4 (绿色)**: 42° → 0° (75-100)
